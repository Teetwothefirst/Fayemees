# FAYEMEES Official Website

FAYEMEES is a premium Nigerian multi-concept lifestyle brand. This repository contains the official website, built to showcase the brand's four distinct arms.

## The Brand
FAYEMEES operates under a unified vision of excellence across:
- **Etana Beauty Salon**: Premium hair and beauty services for women.
- **Eddy's Barber**: Professional grooming and styling for men.
- **Market Place**: Curated products and lifestyle essentials.
- **Feathers, Fins & Flesh (4F's)**: A multi-concept restaurant experience.

## Tech Stack
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Booking**: [Cal.com](https://cal.com/) Integration
- **Typography**: Playfair Display (Headings) & Inter (Body)

## Getting Started

### Prerequisites
- Node.js 18+
- npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables (see `.env.example` - coming soon)
4. Run the development server:
   ```bash
   npm run dev
   ```

## Project Structure
- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI elements, sections, and layout components.
- `src/data`: Structured project data (price lists, service descriptions).
- `src/lib`: Utility functions and shared library configurations.
